# KickMstsc
操作端末の画面解像度に応じて、mstscのオプションを切り替えて実行する<br>
Powershellだと起動が遅いので、Ｃで実装してみた。

## コンパイル方法 
Visual Studio Build Toolsがインストール済み。<br>
cl /D UNICODE /utf-8 /O2 /MT /EHsc .\myrds.cpp /link  /NODEFAULTLIB:libucrt.lib  /DEFAULTLIB:ucrt.lib <br>
<br>



## 動作環境
 Windows<br>
<br>



## 実行方法
そのまま起動させる。
